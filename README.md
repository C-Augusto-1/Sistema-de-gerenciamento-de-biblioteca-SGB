Nova Funcionalidade: Detalhes do Livro com Suporte a Audiobook
Nome da Funcionalidade

Visualização Detalhada de Livros com Integração de Audiobook

Objetivo da Melhoria

Aprimorar a experiência do usuário ao permitir o acesso rápido e intuitivo a informações completas sobre cada livro, incluindo a disponibilidade de audiobooks em plataformas digitais.

Descrição do que foi Implementado

1. Página de Detalhes do Livro
 
* Visualização completa de todas as informações do livro ao clicar no título
* Interface responsiva, com Bootstrap 5
* Exibição organizada de dados do livro e do autor em um único local
  
2. Sistema de Audiobook

* Campo para indicar disponibilidade de audiobook
* Seleção de plataforma (Audible, Spotify, Ubook, Storytel, YouTube, Outros)
* Link direto para ouvir o audiobook na plataforma escolhida
* Badge visual na listagem indicando disponibilidade de audiobook

3. Melhorias no Cadastro

* Campo de descrição/sinopse do livro
* Seção dedicada para informações de audiobook
* Toggle automático de campos relacionados ao audiobook
* Validação de URL para links de audiobook
  
4. Aprimoramentos na Interface
  
* Título do livro clicável na tabela de listagem
* Navegação intuitiva entre páginas
